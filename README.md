
# Job Application Tracker Portal

A Full Stack Job Application Tracker built using React.js, Node.js, Express.js, MongoDB Atlas, and Mongoose.

## Features

- Add Job Applications
- View All Applications
- Edit Existing Applications
- Delete Applications
- Search Jobs by Company Name
- Filter Jobs by Status
- Dashboard Statistics
- MongoDB Database Integration
- Responsive User Interface

## Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

## Installation

### Backend

```bash
cd server
npm install
node index.js
```

### Frontend

```bash
cd client
npm install
npm start
```
